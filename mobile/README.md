# UNTOLD Mobile

Expo monorepo for **UNTOLD Studio Mobile** and **UNTOLD Originals Mobile**.

## Setup

```bash
npm install
```

## Run

```bash
npm run studio      # UNTOLD Studio Mobile
npm run originals   # UNTOLD Originals Mobile
```

See [docs/enterprise-mobile.md](../docs/enterprise-mobile.md) for API surface, offline queues, and production checklist.
